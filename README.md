# print-coveralls

## Installation
```
$ npm install print-coveralls
```

## How to use
```
print-coveralls ./coverage/lcov.info
```


## Output
* Coverage is coveralls's coverage
```
=============================== Coverage summary ===============================
Branches      : 58.1% (423 / 728)
Lines         : 66.22% (806 / 1217)
Coverage      : 63.18% (1229 / 1945)
================================================================================
```
